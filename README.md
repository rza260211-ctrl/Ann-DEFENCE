# Ann Nova Defense (Ann新星防御) 🚀

一个基于 React + Vite + Tailwind CSS 构建的经典《导弹指挥官》风格塔防游戏。

## 🎮 游戏玩法
- **目标**：拦截从天而降的敌方火箭，保护底部的城市和炮台。
- **操作**：点击屏幕任意位置发射拦截导弹。
- **得分**：击毁一个火箭得 20 分，达到 1000 分即获胜。
- **失败条件**：所有 3 座炮台被摧毁。

## 🛠 技术栈
- **前端框架**: React 19
- **构建工具**: Vite 6
- **样式**: Tailwind CSS 4
- **动画**: Motion (Framer Motion)
- **图标**: Lucide React
- **部署**: Vercel

## 🚀 本地开发

1. 克隆仓库:
   ```bash
   git clone <your-repo-url>
   ```
2. 安装依赖:
   ```bash
   npm install
   ```
3. 启动开发服务器:
   ```bash
   npm run dev
   ```

## 🌐 部署到 Vercel

1. 将代码推送到 GitHub。
2. 在 [Vercel 控制台](https://vercel.com) 点击 "Add New" -> "Project"。
3. 导入你的 GitHub 仓库。
4. Vercel 会自动识别 Vite 配置，点击 "Deploy" 即可。
